# Jogo do Número Secreto

Projeto simples desenvolvido com **HTML, CSS e JavaScript** onde o usuário deve adivinhar um número secreto sorteado aleatoriamente entre **1 e 10**. O jogo informa se o número secreto é maior ou menor que o chute, conta as tentativas e permite reiniciar a partida ao acertar.

## Funcionalidades

- Sorteio aleatório de um número entre **1 e 10**
- Verificação do chute do usuário
- Dicas informando se o número secreto é **maior** ou **menor**
- Contador de tentativas
- Botão para iniciar um **novo jogo**
- Reprodução de mensagens em áudio com **Speech Synthesis API** no navegador

## Tecnologias utilizadas

- **HTML5**
- **CSS3**
- **JavaScript**
